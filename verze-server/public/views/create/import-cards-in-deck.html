Import Cards

You can import cards by pasting a textual list of the cards with their phrases, translations, and pronunciations. Follow the example format below.

Example:

Front phrase / pronunciation / example / Back phrase / pronunciation / example
Front phrase / pronunciation / example / Back phrase / pronunciation / example

¡Bienvenido! / BEE-EN-VIN-'EE-DOE / ¡Bienvenido a México! / Welcome! / 'WEL-KUHM / Welcome to Mexico!
¿Cómo estás? / ? / ¿Cómo estás hoy? / How are you? / ? / How are you today?
